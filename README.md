# Share Code

Write and share code in realtime

## Install

```bash
$ yarn install
```

### Env files

```env
PORT=
SESSION_SECRET=

# Github
GH_CLIENT=
GH_SECRET=

# Analytics
GG_ANALYTICS=
```

## Run

### Prod

```bash
$ yarn start
```

### Dev

```bash
$ yarn build:watch

$ yarn server:dev
```

## Features

- [x] Code editor
- [x] Live edit
- [x] Merge on new writer joined in room
- [x] List writers
- [x] Display writer name
- [x] Github Login
- [x] Update users
- [ ] Multi cursors
- [ ] Share button
- [ ] Save on gist
- [ ] Import from gist
